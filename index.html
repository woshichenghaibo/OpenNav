安全警示 / Security Alerts / Оповещения о безопасности

为了保证运行安全，请将站点根目录设置为 /Public 文件夹。
To ensure operational security, set the site root to the /Public folder.
Чтобы обеспечить операционную безопасность, задайте для корневого каталога папку /Public.

如果不进行此操作，请至少确保 /Data 目录不能被直接访问，否则您的 SQLITE 数据库将能够被直接下载。
If you do not do this, at a minimum, make sure that the /Data directory is not directly accessible, or your SQLITE database will be able to be downloaded directly.
Если этого не сделать, как минимум, убедитесь, что каталог /Data недоступен напрямую, иначе базу данных SQLITE можно будет загрузить напрямую.
